# Carpeta de recursos

Guarda aquí imágenes propias, capturas de Canvas, fotos de actividades o archivos visuales que quieras usar dentro del portafolio.

Ejemplo de uso:

```js
photo: "assets/mi-foto.jpg"
```

Para evidencias, puedes enlazar archivos desde `portfolio-data.js`:

```js
evidence: [{ label: "Captura del corto", href: "assets/corto-1.png" }]
```
